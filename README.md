# recomsys

# data source
* itemProps.csv -- 宝贝属性表
* orders.csv -- 订单表

* recom_top10_cluster.csv
* recom_top10_IBCF.csv (Item Based Collaborative Filtering)
* recom_top10_UBCF.csv (User Based Collaborative Filtering)
